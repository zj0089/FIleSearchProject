# File Search Project

This project allows you to search PDF, Word, and Excel files for a list of specific names.

## How to Use the File Search Application

1. **Download the Executable**:
   Go to the [Releases](https://github.com/yourusername/yourrepository/releases) section of this repository and download the latest version of the app (`app.exe` for Windows).

2. **Run the Application**:
   - Simply double-click the downloaded executable to start the app.
   - You do not need to install Python or any other dependencies.

3. **Follow the In-App Instructions**:
   - Use the "Browse" button to select the folder containing files for searching.
   - Enter names manually or load them from a text file by clicking "Load Names from File."
   - Click "Search Files" to begin the search.
